# Hospital Product

A simple hospital management system for tracking patients, appointments, and staff.

## Features

- Patient registration and management
- Appointment scheduling
- Staff directory
- Basic reporting

## Getting Started

1. Clone the repository
2. Install dependencies
3. Run the application
